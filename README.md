# Hello and Welcome to Networks, Signals and Systems Playground! 🌐

Welcome to this educational playground for exploring concepts in Networks, Signals, and Systems! This repository contains interactive visualizers and calculators designed to help you understand fundamental networking and signal processing concepts.

## 📚 What You'll Find Here

This playground includes several interactive tools:

### 🔌 UART Communication Tools
- **`1_UART_SINGLE_CHARACTER_VISUALIZER_Ν`** - Visualize how single characters are transmitted over UART protocol
- **`2_UART_FRAME_VISUALIZER`** - Interactive frame visualization for UART communication

### 📡 Serial Communication
- **`3_SynchronousSerialTransmissionVisualizer`** - Understand synchronous serial data transmission

### 🔀 Multiplexing
- **`4_MultiplexingVisualizer`** - Explore multiplexing concepts in communications

### ⏱️ Pipelining Analysis
- **`5_A_PYTHON_GREEK_CalculateTimePackageTransferPipelining`** - Python script for pipelining calculations (Greek)
- **`5_B_Pipelining Delay Calculator — Networks_English`** - Interactive pipelining delay calculator (English)
- **`5_C_Ypologismos_Pipelining_Kathisterisi_Greek`** - Pipelining delay calculator (Greek)

### 🌐 Network Subnetting
- **`6_Subnet_Calculator`** - Calculate subnet information
- **`7_Subnetting_By_Hosts_or_By_Subnets`** - Advanced subnetting calculator

## 🚀 How to Use

### For HTML Visualizers:
1. Open any `.html` file (files without extension are HTML files) in your web browser
2. Interact with the input fields to see real-time visualizations
3. Most tools include legends and explanations to guide your learning

### For Python Scripts:
1. Ensure you have Python installed on your system
2. Run the Python scripts directly:
   ```bash
   python 5_A_PYTHON_GREEK_CalculateTimePackageTransferPipelining
   ```

## 🎯 Educational Goals

These tools are designed to help you:
- Understand low-level communication protocols like UART
- Visualize how data is transmitted serially
- Learn about multiplexing techniques
- Calculate pipelining delays in network systems
- Master IP subnetting concepts

## 🌍 Language Support

Tools are available in both English and Greek (Ελληνικά) to support different learning preferences.

## 💡 Getting Started

1. Start with the UART visualizers to understand basic serial communication
2. Explore the multiplexing concepts
3. Use the pipelining calculators for performance analysis
4. Practice subnetting with the network calculators

Happy learning! 🎉
